Creating a Dynamic Login Form with Declarative Programming

Imperative Programming
It simply describes the control flow of computation.
-- Classic algorithms - set of instructions - step by step.
Exemples: Object Oriented Programming, Structured Programming.


Declarative Programming
It simply expresses the logic of computation.
-- Instruction by instruction without the whole flow previously defined.
Exemples: Functional programming, typical console environements.
